### git

---

#### 使用命令：

+ git init    创建版本库

+ git add   往里加文件名

+ git status  查看内容

+ git commit -m '随便备注'    提交仓库

+ git log   查看仓库里的内容

+ git clone 克隆  放入ssh / https 地址

+ git reset 还原版本 -- head HEAD(^:表示上一个版本 两个表示 上上版本），也可以使用 哈希值

+ git reflog 查看命令历史

+ ##### 如何创建密钥

  + ssh-keygen -t rsa -C ’输入你的邮箱‘
